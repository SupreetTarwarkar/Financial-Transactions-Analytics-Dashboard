# Financial Transactions Analytics Dashboard

## Project Overview

End-to-end financial analytics solution developed using Microsoft Fabric and Power BI. The dashboard analyzes transaction volume, revenue trends, customer behavior, payment patterns, and geographic performance to generate actionable business insights.

## Objective

- Analyze transaction volume and revenue trends
- Monitor transaction success and failure rates
- Identify customer spending patterns
- Analyze customer segmentation
- Generate business insights from financial transaction data

## Tech Stack

- Microsoft Fabric
- Power BI
- Power Query
- DAX
- Data Modeling
- SQL
- Excel

## Data Source

- Financial Transactions Dataset
- Customer Dataset
- Payment Method Dataset
- Geographic Dataset

## Dashboard Features

### Executive Overview
- Total Revenue
- Total Transactions
- Total Customers
- Average Transaction Value

### Revenue Analytics
- Revenue Trends
- Revenue by Region
- Revenue by Payment Method

### Customer Analytics
- Customer Segmentation
- Customer Spending Patterns
- Geographic Distribution

### Transaction Analytics
- Transaction Success Rate
- Failed Transaction Analysis
- Transaction Volume Trends

## Business Insights

The dashboard helps identify revenue opportunities, customer behavior patterns, transaction bottlenecks, and payment performance trends to support business decision-making.
